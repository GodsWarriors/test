Machine Laguna

  • Description
  • Commentaires

La machine Laguna fournit une précision exceptionnelle, une stabilité et une performance unique. Avec sa performance spéciale, le Laguna fournit des résultats excellents d’application sur le sourcil, l’eye-liner même sur les lèvres. Cette machine idéale est parfaite pour les débutant aussi bien que pour des professionnels. Des accessoires entièrement jetables, hygiéniques avec une application sûre.

  • L’ajustement de vitesse est libre. La puissance de l’appareil peut être réglée selon l’utilisation.
  • Norme CE approuvée.
